# Cheatsheet

(mostly based off [man zshmisc](https://www.mankier.com/1/zshmisc#Simple_Commands_&_Pipelines))

## Simple Commands & Pipelines

A simple command is

```sh
# [OPTIONAL PARAMETERS] COMMAND [ARGUMENTS]
FOO=World zsh -c 'echo Hello $FOO' # Hello World
```
